## 설명

이 포트폴리오는 구직용으로 상업적 목적이 없음을 알려드립니다.<br/><br/>
포트폴리오의 콘셉트는 "ran-up(도움닫기)"을 메인 키워드로 하였습니다.<br/><br/>
상단 gnb 메뉴 클릭시 각 섹션별로 이동할 수 있도록 독특한 사용감을 연출하였으며
크고 선명한 이미지를 활용하여 몰입감 있도록 디자인 하였습니다.
